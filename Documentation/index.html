Template Installation

FTP Upload:

  • Open up your FTP manager and connect to your hosting
  • Browse to required directory (Normally public_html)
  • Upload the files inside mini folder.

Pages

  1. index.html is for homepage.
  2. index-2.html is for homepage 2.
  3. index-3.html is for homepage 3.
  4. index-4.html is for homepage 4.
  5. portfolio.html is for portfolio.
  6. single-project.html is for single project.
  7. blog.html is for blog post grid.

  10. service.html is for service.
